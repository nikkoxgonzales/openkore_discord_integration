
# Openkore Discord Integration
Openkore to Discord
Author: nikkoxgonzales
Version: 1
Language: perl
Date: April 28, 2022
Credits: sctnightcore

Make sure you have the following in your config.txt:
discord_status 1
discord_update 1
discord_delay 2
discord_update_delay 5
discord_token 
discord_channel_id
discord_webhook
Otherwise, this will not work.
